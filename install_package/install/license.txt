PHPCMS X 是基于PHPCMS V9上优化完善而来的，对PHPCMS V9进行了全面的优化和持续的安全修复及维护，并对部分功能做了延伸，PHPCMSX是PHPCMS众多fans的社区维护版。

保留所有官方原版程序功能,保持官方原程序前后台界面,方便二次开发;
主要完善和优化项如下:

修复已知安全漏洞,修复已知BUG
支持后台升级提示和一键更新
全面兼容HTTPS环境，兼容高版本服务器环境(php7.4.9/MariaDB10.4/Nginx1.19.2)
替换并优化了在线编辑器，支持多种插件(代码高亮/涂鸦/表情/百度地图等)
后台附件上传修改为H5上传，前台会员头像H5上传
增加讯飞/百度分词接口(后台配置接口密钥后直接使用)
多处PHP脚本逻辑优化等更多亮点待您亲自发现和体验

更多功能及模块整合不断更新中(预计3.0版更新上线内置模块商店)
